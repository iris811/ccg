Claude Code 환경별 대표 이미지
================================

아래 5개 이미지를 https://claude.com/product/claude-code 에서 저장하세요.

"Use Claude Code where you work" 섹션에서 각 탭을 클릭한 후:
  우클릭 → 이미지를 다른 이름으로 저장

필요한 파일:
  1. terminal.webp   - Terminal 탭 이미지
  2. ide.webp        - IDE 탭 이미지
  3. desktop.webp    - Desktop 탭 이미지
  4. web-ios.webp    - Web and iOS 탭 이미지
  5. slack.webp      - Slack 탭 이미지

저장 위치: 이 폴더 (images/)
